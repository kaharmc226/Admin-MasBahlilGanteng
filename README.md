# Admin-MasBahlilGanteng

Selamat datang di repositori ini! Proyek ini berisi aplikasi **TRAKSI (Transparansi Gizi Nasional)**, sebuah platform pelacakan gizi makanan yang dibangun untuk mendukung program Makanan Bergizi Gratis (MBG) di Indonesia.

## Struktur Direktori

*   [`hackaton/`](./hackaton) - Direktori utama proyek yang berisi aplikasi full-stack TRAKSI (Frontend React + Backend Express/MySQL).

Untuk memulai, silakan masuk ke direktori `hackaton` dan baca instruksi lengkapnya:

```bash
cd hackaton
```

Lihat [README TRAKSI](./hackaton/README.md) untuk informasi instalasi, detail tech stack, akun pengujian, dan cara menjalankan proyek secara lokal.
